# TRADING_RULES_ONEPAGE — 3 сетапа + пресеты (без зоопарка)

Это **шпаргалка** для одинакового исполнения.  
Работаем только от **CORE** уровней (см. `PROTOCOL_CANON_v4_4.md`).

---

## Пресеты
### SWING preset (дефолт)
- Контекст: **W1/D1**
- Сигнал: **H4**
- Вход/точность: **H1** (15m только подтвердить)
- Удержание: **часы–дни**
- TP: **следующий CORE → затем struct/macro**
- SL: **за BUFFER** (не “микро‑стоп”)

### INTRADAY preset (по запросу)
- Контекст: **D1/H4**
- Сигнал: **H1/15m**
- Вход: **15m**
- Удержание: **минуты–часы**
- TP: **ближайший CORE** (частичные тейки быстрее)
- SL: за BUFFER, но буфер обычно уже

---

## Сетап A — BOUNCE от CORE (отбой)
**Когда:**
- цена пришла в **CORE** (допускается короткий прокол BUFFER)
- есть подтверждение: **H1 закрылась обратно** в сторону отбоя (для интра можно 15m)

**Вход:** после подтверждения (или лимит на ретест CORE)  
**SL:** за BUFFER + небольшой запас  
**TP1:** следующий CORE по направлению (частично фиксируем)  
**TP2:** следующий struct/macro  
**Отмена идеи:** нет подтверждения / цена “жует” CORE без реакции

---

## Сетап B — BREAK & RETEST (пробой + ретест)
**Когда:**
- **H4 закрытие** за CORE (желательно и за BUFFER)
- затем ретест CORE с удержанием (не возвращаемся внутрь)

**Вход:** на ретесте CORE  
**SL:** обратно за CORE/BUFFER  
**TP:** следующий CORE → затем следующий

---

## Сетап C — FALSE BREAK (ложный пробой)
**Когда:**
- прокол BUFFER + быстрый возврат в CORE
- подтверждение: **H1 закреп обратно в CORE**

**Вход:** после возврата/ретеста  
**SL:** за экстремум ложняка  
**TP:** ближайший CORE внутрь диапазона → затем следующий

---

## Deriv‑фильтр (простая логика)
Используем как “не лезть в толпу”:
- funding сильно **+** и OI растёт → осторожнее с лонгами, шорты от R работают лучше
- funding сильно **−** и OI растёт → осторожнее с шортами, лонги от S работают лучше
- OI падает → движение часто “выдыхается” → тейки ближе, риск меньше

---

## Риск (жёстко)
- базово **2–4%** на сделку
- **5–7%** только если уровень сильный (≥4) и сетап чистый
- **2 попытки на один уровень**, потом стоп и пауза
- торгуем только если потенциал до следующего уровня **адекватен** (не в пустоте)
