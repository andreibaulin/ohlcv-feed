# README_PROTOCOL — как пользоваться каноном v4.4 (Crypto)

Единственный источник правил — `PROTOCOL_CANON_v4_4.md` в корне проекта.

## Быстрый сценарий (рукопожатие)
1) Ты пишешь: **«Прогноз»** или **«Сетап»**.
2) Я отвечаю **ТОЛЬКО**:
   - 1 строка: **дата/время (Europe/Tallinn) — ID**
   - отдельный блок **«Ссылки»** (одним списком)
   - и **на этом всё** (без цифр/уровней/сценариев).
3) Ты копируешь блок «Ссылки» и **следующим сообщением** присылаешь его обратно.
4) Только после этого я делаю **полный прогноз**:
   - цифры только из `chat_report_latest.md` / `chat_bundle_latest.json`
   - уровни **4S/4R = CORE+BUFFER, без нахлёстов**
   - деривативы тянем **напрямую из Binance FAPI** (ссылки уже в рукопожатии)

## RAW‑режим
Если IRON‑файлы недоступны, но ты хочешь анализ всё равно — напиши **RAW**.  
Тогда цифры берутся из `state_btc_eth_full_latest.json` и live FAPI, и это явно помечается как RAW.
